## Funcion tensorflow

La funcion creada para desplegarse en el cluster fue uan red neuonal sencilla de 5 capas usando keras y TensorFlow. La red neuronal se encarga de clasificar imagenes de n√∫meros escritos a mano en imagenes de resolucion 8x8, el set de entrenamiento usado fue obtenido de la libreria sklearn en la carpeta C√≥digo python en [**number-detector**](number-detector) se encuentra el codigo usado.

### Librerias usadas



### Set de entrenamiento

Se usaron 1000 im√°genes para entrenar, en la carpeta [**example-images**](example-images) hay 100 im√°genes por fuera de las usadas en el entrenamiento para probar la funci√≥n.

```
digits = datasets.load_digits()

images_and_labels = list(zip(digits.images, digits.target))
X=[]
Y=[]
n=np.prod(images_and_labels[0][0].shape) #dimension in line image
for index, (image, label) in enumerate(images_and_labels[:1000]):    
    _X=np.reshape(image,n) # get image in an array
    X.append(_X) # add array image to list of images
    _Y=np.zeros(10) # create the output of the NN
    _Y[label]=1 # ouput nn for images index
    Y.append(_Y) # add the oputput to the list of ouputs
X=np.vstack(X)
Y=np.vstack(Y
```
### Red neuronal

Con keras se creo una red neuronal sencilla con una topolog√≠a [64,16,32,16,10], con una medida de error se error cuadr√°tico medio y usando el algoritmo Adam para la retropropagaci√≥n.

```
topology=[n,16,32,16,10]

model = kr.Sequential()

# A√±adimos la capa 1
model.add(tf.keras.layers.Flatten())

# A√±adimos la capa 2
model.add(kr.layers.Dense(topology[1], activation='relu'))

# A√±adimos la capa 3
model.add(kr.layers.Dense(topology[2], activation='relu'))

# A√±adimos la capa 4
model.add(kr.layers.Dense(topology[3], activation='relu'))

# A√±adimos la capa 5
model.add(kr.layers.Dense(topology[4], activation='sigmoid'))

# Compilamos el modelo, definiendo la funci√≥n de coste y el optimizador.
model.compile(loss='mse', optimizer='adam', metrics=['acc'])

# Y entrenamos al modelo. Los callbacks 
model.fit(X, Y, epochs=100, verbose=0)

model.save('num_detector.model')
```

### Enviar imagenes

Para enviar im√°genes por Openfaas, se codifico las im√°genes en base64 y se env√≠o como string a la funci√≥n, tras esto se decodifico nuevamente y se guardo la informaci√≥n en una imagen dentro del contenedor la cual es cargada y enviada a la red neuronal entrenada

```
binary = os.fsencode(req)
        

image_64_decode = base64.decodebytes(binary) 

result_file = 'image'
with open(result_file, 'wb') as file_handler:
    file_handler.write(image_64_decode)

Image.open(result_file).save(result_file + '.png', 'PNG')
os.remove(result_file)

image = Image.open(result_file + '.png')

image = np.asarray(image)
input_image=np.array([np.reshape(image,np.prod(image.shape)),])

res = get_number(input_image)

```

### Obtener numero imagen

Para Obtener el numero de la imagen se ingresa la imagen cargada a la funci√≥n al modelo ya entrenado.
